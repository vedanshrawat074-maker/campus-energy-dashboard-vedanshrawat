Campus Energy Dashboard by Vedansh Rawat

This project analyzes electricity usage across campus buildings using Python.
It reads raw meter data, cleans it, performs building-wise energy analysis, and generates a dashboard and summary report.

Features

Load and validate CSV meter data

OOP-based modeling for buildings and readings

Building-wise and time-based energy summaries

Dashboard charts showing usage trends

Final text report with key insights

How to Run

Place your CSV files in the data/ folder.

Run the scripts:

python dashboard_solution.py
python final_report_solution.py


Check the output/ folder for the dashboard image and summary report.

Purpose

This project was created as a capstone assignment to demonstrate Python skills in data handling, cleaning, OOP design, visualization, and reporting.
